# Nusantara Language Center - Website

A modern, bilingual educational website for Nusantara Language Center (NLC), an Indonesian language (BIPA) and culture teaching institution.

### Design & UX
- **Premium Modern Design**: Vibrant color palette based on NLC logo (Blue, Green, Orange, Red, Yellow)
- **Smooth Animations**: Micro-interactions, hover effects, and scroll animations
- **Responsive Layout**: Mobile-first design that works on all devices
- **Glassmorphism Effects**: Modern UI with backdrop blur and transparency
- **Accessibility**: WCAG compliant with keyboard navigation and screen reader support

### Bilingual System
- **Auto Language Detection**: Detects browser language (English/Bahasa Indonesia)
- **Manual Language Switcher**: Fixed position toggle for easy switching
- **Persistent Selection**: Saves language preference in localStorage
- **Complete Translation**: All content available in both languages

### Sections
1. **Hero Section**: Eye-catching introduction with gradient backgrounds and CTAs
2. **About NLC**: Institution overview with 4 key advantages
3. **BIPA Programs**: 6 structured levels with pricing and details
4. **Expert Instructors**: 5 certified teachers with international experience
5. **Testimonials**: Student success stories
6. **Contact**: Multiple contact methods and additional services

### Interactive Features
- **Floating WhatsApp Button**: Direct contact with animated pulse effect
- **Smooth Scroll Navigation**: Seamless section transitions
- **Active Section Highlighting**: Navigation updates based on scroll position
- **Mobile Menu**: Responsive hamburger menu for small screens
- **Enrollment Integration**: One-click WhatsApp enrollment with pre-filled messages
